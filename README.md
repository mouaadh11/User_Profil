Flutter User Profile UI

A modern and responsive User Profile UI built with Flutter. This project demonstrates how to create a social media-style profile screen using Flutter widgets such as Stack, Positioned, GridView, BottomNavigationBar, and custom clipping with CustomClipper.

Preview:
Add your application screenshot inside the screenshots folder and update the image path if necessary.

Features :
Beautiful profile header with custom curved cover image
Circular profile picture
User statistics section
Follow button
Gallery and Collection tabs
Responsive image grid gallery
Bottom navigation bar
Clean and organized Flutter code
Custom UI shape using CustomClipper

Technologies Used :
Flutter
Dart
Material Design

Getting Started
Prerequisites

Make sure Flutter is installed:
flutter --version

Assets Configuration :
Ensure your assets are declared in pubspec.yaml:
flutter:
  assets:
    - assets/images/

Learning Objectives :

This project is suitable for Flutter beginners who want to learn:

StatelessWidget vs StatefulWidget
Layout design with Stack and Positioned
GridView implementation
Navigation bars
State management using setState()
Custom UI shapes with CustomClipper
Asset management
